FAQ

Answers before you try Textezi

What the app does, which languages are live, how automation behaves, and where the privacy boundaries sit.

At a glance
Live now Hindi and Punjabi are the current launch languages.
Private by default The core romanization flow stays on-device.
No keyboard swap Textezi works with your normal Android keyboard.
Access model Each install starts with a 30-day trial and a one-time unlock option.

Script change, not translation

Textezi keeps the language the same and rewrites only supported script text into Roman output.

Two automation modes

Auto can rewrite eligible text directly, while Suggest only waits for your confirmation first.

Protected fields stay untouched

Passwords, OTPs, phone fields, and other account-sensitive inputs are intentionally excluded.

Features & Pricing
Textezi converts supported scripts into Roman writing. It does not translate the meaning. It keeps the language the same and changes only the script.
Textezi includes a soft 30-day trial on the device. You can buy the permanent unlock at any time, and after unlock the trial limit no longer matters.
No recurring subscription is part of the current access model. Textezi uses a soft 30-day trial followed by a one-time permanent unlock through Google Play.
Languages & Conversion
Hindi and Punjabi are the active launch languages today. Other languages can be selected and prioritized already, but they stay unchanged until support ships.
Textezi only converts languages that are both supported and enabled. If a language is not enabled, that text should pass through unchanged.
Textezi can already show future languages in Settings, but they remain unchanged until their actual romanization policies are available in the app.
No. Textezi preserves the language and rewrites only the script into Roman output.
Automation
No. Textezi is designed to work with your existing keyboard, especially normal Android typing flows, instead of replacing the keyboard.
Auto rewrites supported text directly in eligible fields. Suggest only shows a short-lived prompt first, and the text changes only if you accept the Textezi suggestion.
The floating bubble is the optional quick control for minimized mode. It lets you reopen Textezi, switch automation mode, or hide the bubble without hunting through settings.
No. Protected inputs such as passwords, OTPs, login fields, phone fields, and similar account-sensitive forms are intentionally excluded from automation.
Privacy & Support
The core workflow is on-device. Text, language settings, saved corrections, and conversion behavior stay local on the phone for the main romanization path.
Textezi is designed to act only on committed text in editable fields that you are actively using, and only after the accessibility disclosure is accepted and the relevant Android permissions are enabled.
Report issues or follow fixes in the Textezi public tracker at sugrowth/Textezi-Public/issues/new. It is the main place for bug reports, workflow feedback, and launch updates.
