# Verified Features

Current verified public product direction:

- Language support is shown in tiers: Validated, Baseline local conversion, and Planned
- Validated support currently covers Hindi, Punjabi, and Urdu
- Textezi changes writing, not meaning
- Textezi is designed to work with the user's existing keyboard workflow
- The floating bubble is an optional quick control for minimized automation management
- The pricing direction is a soft 30-day trial, then a one-time permanent unlock through Google Play
- Optional AI assist is separate from the core purchase and is explicit when used
